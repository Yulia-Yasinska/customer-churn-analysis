# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to identify key factors affecting customer retention.

## Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Visualize key patterns
- Build dashboards for business insights

## Tools
- Python (pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- Tableau

## Key Insights
- Customers with month-to-month contracts have higher churn rates
- Short-term customers are more likely to churn
- Higher monthly charges are associated with increased churn

## Dashboards
Power BI and Tableau dashboards were created to visualize key metrics and trends.
